Привет. Это новый проект Yachtworth — мобильное приложение для оценки яхт, отдельный бренд от существующей платформы PDYE. Я владелец, не технический человек, говорю по-русски. Общайся со мной простыми словами без жаргона.

ВАЖНО: я уже всё спланировал с другим агентом в другом Replit-проекте. Не задавай мне те же вопросы заново. Вот всё, что уже решено:

=== ПРОЕКТ ===
Название: Yachtworth
Что это: мобильное приложение (iOS + Android) для AI-оценки яхт. Целевая аудитория: владельцы яхт + брокеры/сюрвейеры. Standalone бренд, но с кросс-сслыкой на PDYE для оценок > €5M.

=== БРЕНДИНГ ===
Цвета: deep navy #0B1E3F + champagne gold #C9A961
Стиль: люксовый минимализм
Шрифты: подобрать под luxury (что-то вроде Playfair Display + Inter)
Логотип и иконка приложения уже сгенерированы — лежат у меня, я их пришлю когда понадобятся.

=== СТЕК ===
- Frontend: Expo React Native (один код для iOS+Android). Используй skill `expo`.
- Backend: Express API (новый артефакт `artifacts/yachtworth-api`)
- База: Supabase (новый отдельный проект `yachtworth-prod`, регион Frankfurt — у меня уже создан, ключи дам через защищённую форму)
- Auth: Clerk (Apple Sign-In обязательно для App Store + Google Sign-In + Email)
- Подписки: RevenueCat (используй skill `revenuecat`)
- Деплой: Render (отдельный сервис, не трогать существующий PDYE на Render)

=== МОНЕТИЗАЦИЯ ===
- Базовый тариф: €19.99/месяц
- Pro тариф: €49.99/месяц (история, PDF экспорт, безлимит оценок)
- 7-дневный бесплатный пробный период
- Бесплатный режим: 1 оценка в месяц без сохранения истории

=== СВЯЗЬ С PDYE ===
Логотип Yachtworth чистый, без упоминаний PDYE. Но внутри приложения:
- В onboarding: «by the team behind PDYE»
- В Settings/About: блок «Powered by PDYE»  
- Для оценок > €5M или Quick-sale тип: баннер с предложением перейти на PDYE для брокерского сопровождения

=== ПЛАН РАБОТЫ (7 дней разработки) ===
День 1: Каркас приложения + design system + табы (Главная / История / Профиль) + создание схемы базы (users, valuations, subscriptions)
День 2: Auth через Clerk (Apple + Google + Email)
Дни 3-4: Форма оценки яхты + AI-эстимация + экран результата с диапазоном цены/графиком/PDF
День 5: История оценок + Профиль + Settings с блоком «Powered by PDYE»
День 6: RevenueCat + paywall + 7-day trial + ограничения для бесплатных
День 7: Скриншоты + описания для App Store/Google Play + Privacy Policy + Terms + сборка билдов через EAS Build

=== НЕ ВКЛЮЧАЕМ В v1.0 ===
Push-уведомления, многоязычность, корпоративные аккаунты, загрузка фото яхт, сайт-визитка. Это всё v1.1+.

=== AI-ЛОГИКА ОЦЕНКИ ===
В существующем PDYE есть рабочая AI-валюация яхт (использует OPENAI_API_KEY). Логика — серверная, на Express, с базовой ценой по длине/году/верфи/состоянию + множители (condition multiplier: New 1.05 / Excellent 1.00 / Good 0.93 / Fair 0.83 / Needs Refit 0.70 / Project 0.50, distressed -20%, quick-sale -30%). Можно начать с похожей логики и улучшать. Я смогу прислать референс из PDYE если нужно.
