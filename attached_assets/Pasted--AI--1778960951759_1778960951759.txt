Создай профессиональное мобильное AI-приложение для расчета рентабельности яхт, сдаваемых в чартер. Приложение должно работать как инвестиционный и аналитический инструмент для владельцев яхт, брокеров и инвесторов.

Технологии:

* Frontend: React + TypeScript + Tailwind
* Mobile-first дизайн
* Backend: Supabase
* Database: PostgreSQL
* Authentication: Supabase Auth
* Charts: Recharts
* AI integration: OpenAI API
* Hosting: Replit Deployments

Основная идея:
Пользователь вводит параметры яхты, а AI автоматически рассчитывает:

* доходность
* расходы
* ROI
* загрузку яхты
* окупаемость
* прогноз прибыли
* прогноз остаточной стоимости яхты
* рекомендации по регионам и сезонам

Сделай приложение современным, premium-класса, в стиле Bloomberg Terminal / modern yacht tech platform.

3. AI CHARTER CALCULATOR

AI должен рассчитывать:

GROSS REVENUE:
Gross Revenue = Daily Rate × Booked Days

OPERATING PROFIT:
Operating Profit = Revenue - Expenses - Broker Commissions

ROI:
ROI = (Net Profit + Residual Value - Depreciation) / Initial Investment
